# GeminiTelegramBot: рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў ­ЪЏа№ИЈ

рЃћрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ рЃљрЃдрЃгрЃћрЃарЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃљрЃДрЃћрЃюрЃЮрЃЌ рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ GeminiTelegramBot рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃџрЃЮрЃЎрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў. ­Ъњ╗

## рЃгрЃўрЃюрЃљрЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў РюЁ

рЃАрЃљрЃюрЃљрЃЏ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃЌ, рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЦрЃЋрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў: ­ЪЉЄ

- **Python 3.13+:** рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃћрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ Python-рЃўрЃА рЃБрЃљрЃ«рЃџрЃћрЃАрЃў рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃЌ. рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЮрЃЌ [Python-рЃўрЃА рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃЊрЃљрЃю](https://www.python.org/downloads/). ­ЪљЇ
- **Telegram рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў рЃЊрЃљ рЃЉрЃЮрЃбрЃўрЃА рЃбрЃЮрЃЎрЃћрЃюрЃў:** рЃЌрЃЦрЃЋрЃћрЃю рЃЊрЃљрЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ Telegram рЃЉрЃЮрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ [BotFather](https://core.telegram.org/bots#botfather)-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃЊрЃљ рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў API рЃбрЃЮрЃЎрЃћрЃюрЃў. ­Ъцќ­ЪћЉ
- **Google Gemini API рЃЎрЃџрЃљрЃЋрЃўрЃерЃў:** рЃЏрЃўрЃўрЃдрЃћрЃЌ API рЃЎрЃџрЃљрЃЋрЃўрЃерЃў [Google AI Studio](https://aistudio.google.com/). рЃћрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. Рюе­Ъќ╝№ИЈ­Ъјц
- **Hugging Face API рЃЎрЃџрЃљрЃЋрЃўрЃерЃў:** рЃЏрЃўрЃўрЃдрЃћрЃЌ API рЃЎрЃџрЃљрЃЋрЃўрЃерЃў [Hugging Face](https://huggingface.co/settings/tokens). рЃћрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА Retrieval-Augmented Generation (RAG) рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃЌ. ­ЪцЌ­ЪћЉ
- **рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб рЃЎрЃљрЃЋрЃерЃўрЃарЃў:** рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ рЃЊрЃљ AI рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃЌрЃљрЃю рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ. ­Ъїљ

## 1. рЃАрЃљрЃфрЃљрЃЋрЃўрЃА рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ ­ЪЊЦ

рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃљрЃфрЃљрЃЋрЃў:

```sh
git clone <рЃЌрЃЦрЃЋрЃћрЃюрЃў-рЃАрЃљрЃфрЃљрЃЋрЃўрЃА-рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў>
cd GeminiTelegramBot
```

## 2. Pipenv-рЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ ­ЪЊд

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА [Pipenv](https://pipenv.pypa.io/en/latest/)-рЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ. рЃЌрЃБ Pipenv рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃљрЃЊ. **рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ:** рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃАрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ. Рџа№ИЈ

```sh
pip install --user pipenv
```

Pipenv-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў (рЃЌрЃБ рЃБрЃЎрЃЋрЃћ рЃљрЃа рЃ«рЃљрЃарЃЌ) рЃЊрЃљ рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ. Pipenv рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА. РюЁ

```sh
pipenv install
```

рЃћрЃА рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃЊрЃљ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ `Pipfile`-рЃерЃў, рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃа рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў.

## 3. рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ (`.env` рЃцрЃљрЃўрЃџрЃў) ­ЪћЉ

рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃћрЃБрЃџ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃцрЃљрЃўрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ `.env` рЃЊрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃ«рЃљрЃќрЃћрЃЉрЃў. рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ placeholder рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃарЃћрЃљрЃџрЃБрЃарЃў API рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃЌрЃљ рЃЊрЃљ рЃЎрЃџрЃљрЃЋрЃўрЃерЃћрЃЉрЃўрЃЌ. РюЇ№ИЈ

```env
BOT_TOKEN="рЃЌрЃЦрЃЋрЃћрЃюрЃў_TELEGRAM_BOT_TOKEN"
GEMINI_API_KEY="рЃЌрЃЦрЃЋрЃћрЃюрЃў_GEMINI_API_KEY"
MODEL_NAME="gemini-2.5-flash-preview-05-20"
HUGGING_FACE_API_KEY="рЃЌрЃЦрЃЋрЃћрЃюрЃў_HUGGING_FACE_API_KEY"
HUGGING_FACE_MODEL="google/gemma-2b-it"
```

- **`BOT_TOKEN`:** рЃЌрЃЦрЃЋрЃћрЃюрЃў Telegram рЃЉрЃЮрЃбрЃўрЃА рЃбрЃЮрЃЎрЃћрЃюрЃў. ­Ъцќ
- **`GEMINI_API_KEY`:** рЃЌрЃЦрЃЋрЃћрЃюрЃў Google Gemini API рЃЎрЃџрЃљрЃЋрЃўрЃерЃў. Рюе
- **`MODEL_NAME`:** Google Gemini рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў. ­ЪДа
- **`HUGGING_FACE_API_KEY`:** рЃЌрЃЦрЃЋрЃћрЃюрЃў Hugging Face API рЃЎрЃџрЃљрЃЋрЃўрЃерЃў. ­ЪцЌ
- **`HUGGING_FACE_MODEL`:** Hugging Face рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА ID рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃЌ). ­Ъцќ

## 4. рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ РќХ№ИЈ

рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ Pipenv-рЃўрЃА `run` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ. рЃћрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃЉрЃЮрЃбрЃў рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ рЃАрЃгрЃЮрЃа рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў:

```sh
pipenv run python bot_geo_v1.py
```

рЃЉрЃЮрЃбрЃў рЃљрЃ«рЃџрЃљ рЃБрЃюрЃЊрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃћрЃА рЃЊрЃљ рЃЏрЃќрЃљрЃЊ рЃўрЃДрЃЮрЃА Telegram-рЃерЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ. ­ЪјЅ

## 5. рЃАрЃљрЃБрЃЉрЃарЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ (рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа) ­ЪњЙ

рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ `user_conversations.csv` рЃцрЃљрЃўрЃџрЃерЃў рЃЊрЃљ рЃњрЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃЉрЃЮрЃбрЃЏрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃўрЃА RAG-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА, рЃЏрЃЮрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЌ рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃўрЃЏрЃљрЃЋрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў, рЃАрЃљрЃЊрЃљрЃф `bot_geo_v1.py` рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃА. рЃЉрЃЮрЃбрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃћрЃфрЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА рЃАрЃбрЃљрЃарЃбрЃљрЃърЃўрЃА рЃЊрЃарЃЮрЃА. ­Ъћё
