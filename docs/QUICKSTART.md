# GeminiTelegramBot: рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ Рюе

рЃћрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃЌ GeminiTelegramBot-рЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ. ­Ъџђ

## рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

1.  **рЃАрЃљрЃфрЃљрЃЋрЃўрЃА рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ:** ­ЪЊЦ
    рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў:

    ```sh
    git clone <рЃЌрЃЦрЃЋрЃћрЃюрЃў-рЃАрЃљрЃфрЃљрЃЋрЃўрЃА-рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў>
    cd GeminiTelegramBot
    ```

2.  **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ Pipenv-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ:** ­ЪЊд
    рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА [Pipenv](https://pipenv.pypa.io/en/latest/) -рЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ. рЃЌрЃБ Pipenv рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃљрЃЊ ( рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃљрЃа рЃ«рЃљрЃарЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў): Рџа№ИЈ

    ```sh
    pip install --user pipenv
    ```

    рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ:

    ```sh
    pipenv install
    ```

    рЃћрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЊрЃљ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃА `Pipfile`-рЃерЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџ рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљрЃА. РюЁ

3.  **`.env` рЃцрЃљрЃўрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ:** ­ЪћЉ
    рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃћрЃБрЃџ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃерЃўрЃюрЃљрЃљрЃарЃАрЃўрЃЌ. рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ placeholder рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃарЃћрЃљрЃџрЃБрЃарЃў рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃЎрЃџрЃљрЃЋрЃўрЃерЃћрЃЉрЃўрЃЌ. РюЇ№ИЈ

    ```env
    BOT_TOKEN="рЃЌрЃЦрЃЋрЃћрЃюрЃў_TELEGRAM_BOT_TOKEN"
    GEMINI_API_KEY="рЃЌрЃЦрЃЋрЃћрЃюрЃў_GEMINI_API_KEY" # рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃАрЃБрЃарЃљрЃЌрЃўрЃА/рЃљрЃБрЃЊрЃўрЃЮрЃАрЃЌрЃЋрЃўрЃА
    MODEL_NAME="gemini-2.5-flash-preview-05-20" # рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃАрЃБрЃарЃљрЃЌрЃўрЃА/рЃљрЃБрЃЊрЃўрЃЮрЃАрЃЌрЃЋрЃўрЃА
    HUGGING_FACE_API_KEY="рЃЌрЃЦрЃЋрЃћрЃюрЃў_HUGGING_FACE_API_KEY" # рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃбрЃћрЃЦрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА (RAG)
    HUGGING_FACE_MODEL="google/gemma-2b-it" # рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА (RAG)
    ```

    - **`BOT_TOKEN`:** рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ [BotFather](https://core.telegram.org/bots#botfather)-рЃњрЃљрЃю Telegram-рЃерЃў. ­Ъцќ
    - **`GEMINI_API_KEY`:** рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ [Google AI Studio](https://aistudio.google.com/)-рЃЊрЃљрЃю (рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА). ­Ъќ╝№ИЈ­Ъјц
    - **`MODEL_NAME`:** Google Gemini рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, `gemini-2.5-flash-preview-05-20`). ­ЪДа
    - **`HUGGING_FACE_API_KEY`:** рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ [Hugging Face](https://huggingface.co/settings/tokens)-рЃЊрЃљрЃю (рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА RAG-рЃўрЃЌ). ­ЪцЌ
    - **`HUGGING_FACE_MODEL`:** Hugging Face рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА ID рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃљрЃЊ `google/gemma-2b-it`). ­Ъњг

4.  **рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ:** РќХ№ИЈ
    рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЉрЃЮрЃбрЃў Pipenv рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў:

    ```sh
    pipenv run python bot_geo_v1.py
    ```

    рЃЉрЃЮрЃбрЃў рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃЏрЃќрЃљрЃЊ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ Telegram-рЃерЃў. ­ЪјЅ

5.  **рЃАрЃљрЃБрЃЉрЃарЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ (рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа):** ­ЪњЙ
    рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў `user_conversations.csv` рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃњрЃАрЃБрЃарЃЌ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ RAG-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃЏрЃЊрЃћ, рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃцрЃљрЃўрЃџрЃў рЃАрЃгрЃЮрЃа рЃљрЃЊрЃњрЃўрЃџрЃљрЃАрЃљрЃљ (рЃўрЃЏрЃљрЃЋрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў, рЃАрЃљрЃЊрЃљрЃф `bot_geo_v1.py`). рЃЉрЃЮрЃбрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃћрЃфрЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА (рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ in-memory Chroma-рЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЏрЃЪрЃљрЃЏрЃљрЃЊ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ). ­Ъћё
