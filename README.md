# ­ЪЄг­ЪЄфРюе Gemini Telegram Bot: рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ (v1.1) Рюе­ЪЄг­ЪЄф

­ЪЉІ рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ! рЃћрЃА рЃљрЃарЃўрЃА Telegram рЃЉрЃЮрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃљрЃЊ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Telegram Bot API-рЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃЌрЃљрЃю рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃЊрЃљрЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃўрЃЏрЃБрЃерЃљрЃЮрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃћрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃЌрЃљрЃю, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃљрЃарЃЌрЃБрЃџрЃў, рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃљ.

## ­Ъџђ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

- РюЁ **`/start` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:** рЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃљрЃЌ рЃЏрЃўрЃАрЃљрЃАрЃљрЃџрЃЏрЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃўрЃЌ рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ.
- РЮЊ **`/help` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:** рЃњрЃљрЃгрЃЋрЃЊрЃўрЃЌ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЉрЃЮрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ.
- ­Ъњг **рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ:** рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (AI рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃљрЃЊрЃњрЃўрЃџрЃў).
- РџЎ№ИЈ **рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃЏрЃърЃбрЃћрЃЉрЃў:** рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа рЃърЃарЃЮрЃЏрЃърЃбрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃърЃљрЃАрЃБрЃ«рЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў AI рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃЊрЃљрЃю.

## ­ЪЏа№ИЈ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

1.  **рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃА рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ:**

    ```bash
    git clone <repository_url>
    cd GeminiTelegramBot
    ```

2.  **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ:**

    рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЦрЃЋрЃЌ Python. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў pip-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ:

    ```bash
    pip install -r requirements.txt
    ```

3.  **Telegram рЃЉрЃЮрЃбрЃўрЃА рЃбрЃЮрЃЎрЃћрЃюрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ:**

    рЃљрЃўрЃдрЃћрЃЌ рЃЉрЃЮрЃбрЃўрЃА рЃбрЃЮрЃЎрЃћрЃюрЃў BotFather-рЃўрЃАрЃњрЃљрЃю Telegram-рЃерЃў. рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ рЃћрЃА рЃбрЃЮрЃЎрЃћрЃюрЃў рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў (`environment variable`) рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ `TELEGRAM_BOT_TOKEN`.

    - ­ЪќЦ№ИЈ **Windows-рЃќрЃћ:**

      ```bash
      $env:TELEGRAM_BOT_TOKEN="YOUR_BOT_TOKEN"
      # рЃљрЃю рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ: System Properties > Environment Variables
      ```

    - ­ЪЇј­ЪљД **macOS/Linux-рЃќрЃћ:**

      ```bash
      export TELEGRAM_BOT_TOKEN="YOUR_BOT_TOKEN"
      # рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА shell рЃърЃарЃЮрЃцрЃўрЃџрЃА (.bashrc, .zshrc, рЃЊрЃљ рЃљ.рЃе.) рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
      ```

4.  **рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ:**

    рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ Python рЃАрЃЎрЃарЃўрЃърЃбрЃў:

    ```bash
    python bot_geo_v1.py
    ```

## ­ЪДа AI рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ

рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ `bot_geo_v1.py` рЃљрЃЏрЃЪрЃљрЃЏрЃљрЃЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЊрЃњрЃўрЃџрЃА AI рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃЉрЃЮрЃбрЃЏрЃљ рЃўрЃЏрЃБрЃерЃљрЃЮрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃЏрЃљ, рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃћрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА (рЃЏрЃљрЃњ. Gemini, GPT, рЃЊрЃљ рЃљ.рЃе.) рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃљ. рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ `handle_message` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃЮрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃарЃЕрЃћрЃБрЃџрЃў AI рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА API.

`SYSTEM_PROMPT_GEO` рЃфрЃЋрЃџрЃљрЃЊрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃърЃарЃЮрЃЏрЃърЃбрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў AI рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃљрЃЊ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА, рЃњрЃљрЃЏрЃљрЃарЃЌрЃБрЃџрЃў рЃЊрЃљ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃџрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ. рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃюрЃў AI рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃљрЃЏ рЃљрЃю рЃЏрЃАрЃњрЃљрЃЋрЃА рЃърЃарЃЮрЃЏрЃърЃбрЃА.

## ­ЪЊќ Memory Bank (рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЉрЃљрЃюрЃЎрЃў)

`memory-bank/` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Cursor AI рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА, рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃЊрЃљ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃљрЃЊ. рЃћрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ Cursor-рЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪЎї рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЋрЃўрЃбрЃљрЃюрЃЮрЃЌ рЃгрЃЋрЃџрЃўрЃџрЃў

рЃљрЃЦ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃАрЃ«рЃЋрЃћрЃЉрЃА рЃгрЃЋрЃџрЃўрЃџрЃўрЃА рЃерЃћрЃбрЃљрЃюрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў.
